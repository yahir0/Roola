# Roola ブランドアセット

Roola のアプリアイコン／シンボルマークの定義とソースを置く。

## コンセプト

デザインシステム Polaris（ADR-0038）の機能主義に準拠したシンボル。
アプリの核「目的地へ素早く移動する」と、Polaris（北極星 ＝ 不動の導きの星）
を、装飾を排した幾何学だけで表す。

- **中央の円環** — ユーザーの現在地・環境に左右されない「不変の基準点」。
  Polaris（北極星）＝ 方向を指し示す不動の導き。
- **東西南北の三角形** — 「あらゆる場所・操作」へ導く 4 方向の指針。
  どこへでも行ける、を表す矢印。アプリ名 Roola の語源である移動呪文
  「ルーラ」（任意の地点へ一瞬で移動する）に通じる。
- **シンボル全体の形** — 円環から 4 方向へ突き出た三角形は、船を操る
  「舵（ヘルム）」を象る。北極星で方角を定め、舵でそこへ向かう
  ── アプリそのものの体験を一つの図形に重ねている。
- グラデーション・影・装飾モチーフは持たない（ADR-0038 D13）。

## 仕様

| 項目 | 値 |
| --- | --- |
| 背景 | グラファイト `#121317` の角丸スクエア（macOS グリッド 824/1024、角丸 R=184） |
| シンボル | オフホワイト `#ECEFF2`（ヘッダーの `ROOLA` ワードマークと同色） |
| 構造 | 外周円環＋中央円環＋東西南北の三角形（中心の点は持たない） |

## ファイル

- `roola_icon.svg` — シンボルマークのベクターソース（編集の起点）。
- `generate_app_icon.py` — `roola_icon.svg` と同じ幾何学を Pillow で描画し、
  `macos/Runner/Assets.xcassets/AppIcon.appiconset/` の PNG 7 サイズ
  （16〜1024px）を生成する。SVG レンダラに依存しない。

## アイコンの再生成

```sh
python3 branding/generate_app_icon.py
```

寸法・色を変えるときは `generate_app_icon.py` 冒頭のデザイントークンと
`roola_icon.svg` の双方を更新する（両者は同じ幾何学を表す）。macOS の
Dock アイコンはビルド時に焼き込まれるため、変更後はアプリの再ビルドが必要。
