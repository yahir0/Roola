# Roola ブランドアセット

Roola のアプリアイコン／シンボルマークの定義とソースを置く。

## コンセプト

アプリの核「目的地へ素早く移動する」を、**翼**と**フォルダ**で表す。

- **大きな白い翼** — トレードマーク。広く包み込む形で「安心感」と
  「遠くまで飛べる」を表す。アプリ名 Roola の語源である移動呪文「ルーラ」
  （任意の地点へ一瞬で移動する）と、DQ の黄金の鳥ラーミア（世界を飛び回る
  移動手段）に通じる。渡り鳥が星を頼りに航行する＝「導き」のニュアンスも持つ。
- **ゴールドのフォルダ** — エクスプローラ＝アプリの核機能。「目的地（登録した
  場所）」そのもの。翼のカーブがフォルダを抱く構図で、保護・安心を表す。
- 地はグラファイト。アクセントはゴールド（Polaris の暖色アクセント・ADR-0038）。

旧シンボル（舵／コンパス＝北極星 Polaris の幾何モチーフ）からの刷新の経緯は
ADR-0053 を参照。

## 仕様

| 項目 | 値 |
| --- | --- |
| 背景 | グラファイトの角丸スクエア（macOS グリッド 824/1024、角丸 R=184、外側透明） |
| シンボル | 白い翼（オフホワイト系）＋ ゴールド `#D0A341` のフォルダ |

## ファイル

- `roola_icon_master.png` — ブランドシンボルのマスター画像（1024×1024・角丸
  スクエア本体＋外側透明）。**編集・再生成の起点**。
- `generate_app_icon.py` — マスターを各サイズへ縮小し、
  `macos/Runner/Assets.xcassets/AppIcon.appiconset/` の PNG 7 サイズ
  （16〜1024px）を生成する。元アートを差し替える場合は `_build_master()` で
  マスターを作り直す。

> 旧 `roola_icon.svg`（舵モチーフのベクターソース）は ADR-0053 でシンボルを
> 刷新したため廃止した。現行シンボルはラスター（マスター PNG）が起点。

## アイコンの再生成

```sh
python3 branding/generate_app_icon.py
```

macOS の Dock アイコンはビルド時に焼き込まれるため、変更後はアプリの再ビルドが
必要。
