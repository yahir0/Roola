# Claude Skills Launcher 動作確認用スキル

これは Claude Skills Launcher 本体の動作確認専用スキルです。並行実行・
キャンセル・閉じる・確認ダイアログなどの UI 挙動を、本物の Skill を
動かす副作用なしに検証するために使います。

## やること（これだけ）

**Bash ツールで以下のコマンドを一度だけ実行してください。** それ以外の
ことは一切しないでください（ファイル変更・Git 操作・Web 取得・
追加のプロンプト確認など）。timeout は 90 秒以上にしてください。

```bash
echo "test-launcher-pulse start: $(date '+%H:%M:%S')"
for i in $(seq 1 12); do
  echo "[$i/12] $(date '+%H:%M:%S') heartbeat"
  sleep 5
done
echo "test-launcher-pulse done: $(date '+%H:%M:%S')"
```

実行が終わったら「テスト完了しました」と 1 行だけ返答してセッションを
終えてください。途中で何か聞かれても処理せず、追加の tool 呼び出しは
行わないでください。

## このスキルが何のためのものか

- ランチャーから `claude` プロセスを PTY 上で起動できることを確認
- ターミナル出力（毎 5 秒のハートビート）が描画されることを確認
- 約 1 分間「running」状態が続く間に
  - ホームへ戻って別セッションを並行起動できることを確認
  - chip 列・アイコンバッジが状態に応じて更新されることを確認
  - 「キャンセル」で PTY が止まることを確認
- 終了後に「閉じる」でセッションが破棄できることを確認

実 Skill のように API コストやファイル副作用が出ないので、開発中の
動作確認は本スキルだけで行うことを推奨します。
