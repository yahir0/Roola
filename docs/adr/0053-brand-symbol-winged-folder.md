# ADR-0053: ブランドシンボルを舵から「翼＋フォルダ」へ刷新する

- **Status**: Accepted
- **Date**: 2026-05-22

## Context

従来のアプリアイコン／シンボルは、北極星 Polaris（不動の導き）と船の舵
（ヘルム）を、中央円環＋外周円環＋東西南北の三角形矢印で表す幾何モチーフ
だった（branding/README.md 旧版、[ADR-0038] の機能主義に準拠）。

しかしこのモチーフは、コンセプトは支持される一方で**見た目が威圧的**という
課題があった。具体的には:

- 中央円＋外周環＋4 方位マークの構図が、**照準器（レティクル）／監視カメラの
  目**に読まれる。
- 尖った 4 方向の三角形が**手裏剣／刃物**に読まれる。

色（ゴールド化・2 トーン化）や角の処理（先端の丸め・凹みの調整）では「目・
照準・棘」のゲシュタルトが解消せず、中心を埋めると今度は「目」が強まる、と
いう構造的なジレンマがあった。中心円＋放射という骨格自体が威圧感の温床だった。

## Decision

ブランドシンボルを、幾何モチーフ（舵／コンパス）から
**「大きな翼がフォルダを抱く」具象モチーフ**へ刷新する。

- **大きな白い翼** — トレードマーク。広く包み込む形で「安心感」と「遠くまで
  飛べる」を表す。語源「ルーラ」（瞬間移動）、DQ の黄金の鳥ラーミア（移動
  手段）、渡り鳥の星による航行（導き）に通じる。
- **ゴールドのフォルダ** — エクスプローラ＝アプリの核機能。「目的地（登録した
  場所）」。翼のカーブがフォルダを抱く構図で保護・安心を表す。
- 配色は [ADR-0038] の枠内（グラファイト地＋ゴールドアクセント＋オフホワイト）。

コンセプト（目的地への素早い移動・導き）は維持し、それを表す**視覚メタファー
だけを差し替える**。北極星 Polaris の語は引き続きデザインシステム名として残る。

## Consequences

- アイコンは幾何ベクターではなく**ラスター（イラスト）由来**になった。マスター
  画像 `branding/roola_icon_master.png`（1024・角丸スクエア＋外側透明）を起点と
  し、`branding/generate_app_icon.py` が macOS グリッド（824/1024・角丸 R=184）
  へ整形して 7 サイズを生成する。
- 旧 `branding/roola_icon.svg`（舵モチーフ）は廃止した。
- 翼に微細な陰影が乗るため、[ADR-0038] の「グラデーション禁止」とは厳密には
  一致しない。アプリ内 UI のフラット原則は維持しつつ、**アプリアイコンは例外**と
  して扱う（OS アイコンは多少の陰影・質感が許容される慣習に従う）。
- メニュー／アプリ内で使う**単色シルエット版**は別途必要（未着手）。
- 既存の機能主義の説明（branding/README.md 旧版の「装飾を排した幾何学」）は
  本 ADR で上書きされる。
