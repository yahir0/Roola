## ADDED Requirements

### Requirement: ノートパッドパネルの開閉

ノートパッドはワークスペース右下に重ねて表示するフローティングパネルとして提供され、ヘッダのトグルアイコンで開閉できる。

#### Scenario: ヘッダのアイコンでパネルを開く

- **WHEN** ユーザーがヘッダのノートパッドアイコンを、パネルが閉じた状態でクリックする
- **THEN** ワークスペース領域の右下にノートパッドパネルが表示され、本文入力欄にフォーカスが当たる

#### Scenario: ヘッダのアイコンでパネルを閉じる

- **WHEN** ユーザーがヘッダのノートパッドアイコンを、パネルが開いた状態でクリックする
- **THEN** ノートパッドパネルが非表示になる

#### Scenario: パネルの閉じるボタンで閉じる

- **WHEN** ユーザーがパネルのヘッダにある閉じるボタンをクリックする
- **THEN** ノートパッドパネルが非表示になる

#### Scenario: パネルはワークスペースレイアウトに影響しない

- **WHEN** ノートパッドパネルを開閉する
- **THEN** 3 ペインの分割・スプリッタ比率・タブ構成は変化せず、パネルはその上に重ねて表示される

### Requirement: ノートパッドの本文編集

ノートパッドパネルは本文の入力と、標準的なテキスト編集ショートカットによる操作ができる。

#### Scenario: 本文を入力する

- **WHEN** パネルの本文入力欄に文字を入力する
- **THEN** 入力した文字が本文として表示・保持される

#### Scenario: 標準の編集ショートカットが使える

- **WHEN** 本文入力欄にフォーカスがある状態でコピー / ペースト / カット / 全選択 / 取り消し（⌘C / ⌘V / ⌘X / ⌘A / ⌘Z）を行う
- **THEN** それぞれ標準のテキスト編集動作が行われる

### Requirement: 行番号ルーラ

ノートパッドパネルは本文入力欄の左端に行番号ルーラを表示する。

#### Scenario: 論理行ごとに行番号を表示する

- **WHEN** 本文が複数の論理行（改行区切り）を含む
- **THEN** 各論理行の先頭に対応する行番号（1 始まり）がルーラに表示される

#### Scenario: 折り返し時も行番号がずれない

- **WHEN** ある論理行が入力欄の幅で複数表示行に折り返される
- **THEN** 行番号は折り返し後の論理行の先頭位置に表示され、後続行の番号がずれない

### Requirement: ノートパッド本文の永続化

ノートパッドの本文は永続化され、アプリ再起動後も復元される。

#### Scenario: 本文を保存する

- **WHEN** ユーザーが本文を編集する
- **THEN** 本文は `<appSupport>/notepad.json` に永続化される

#### Scenario: 再起動後に本文を復元する

- **WHEN** アプリを再起動し、ノートパッドパネルを開く
- **THEN** 前回終了時の本文が復元されて表示される

#### Scenario: パネルを閉じても本文を保持する

- **WHEN** 本文を編集した後にパネルを閉じ、再度開く
- **THEN** 直前の本文が保持されて表示される

#### Scenario: 永続データが無い / 壊れている場合

- **WHEN** `notepad.json` が存在しない、または JSON として不正
- **THEN** ノートパッドは空の本文で開始し、エラーにならない
